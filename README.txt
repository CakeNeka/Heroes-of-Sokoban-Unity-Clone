                          ==== Heroes Of Sokoban ====

Heroes of Sokoban is a puzzle game series by Jonah Ostroff, I made this clone in
Unity to practice game development. It's really simple but someone may find it 
useful.

- Heroes of Sokoban https://sites.math.washington.edu/~ostroff/puzzles/Heroes_of_Sokoban.html
- Heroes of Sokoban II: Monsters https://sites.math.washington.edu/~ostroff/puzzles/Heroes_of_Sokoban_II_Monsters.html
- Heroes of Sokoban III: The Bard  https://www.puzzlescript.net/play.html?p=7072276
